# 翻译工具 / Translation Tool

基于 Qt 6.9.2 开发的桌面翻译应用程序，支持中英互译、历史记录管理、本地缓存等功能。

## 功能特性

- ✅ 中英互译（支持英→中、中→英）
- ✅ 多释义展示（显示单词的多个可能含义）
- ✅ 例句展示
- ✅ 查询历史记录管理（查看、搜索、删除）
- ✅ 本地 SQLite 缓存（提高查询速度）
- ✅ 历史记录搜索过滤（使用 ProxyModel）

## 技术栈

- **框架**：Qt 6.9.2
- **语言**：C++17
- **数据库**：SQLite
- **网络**：Qt Network (HTTP)
- **架构**：Model/View + ProxyModel

## 编译要求

- Qt 6.9.2 或更高版本
- MinGW 64-bit 编译器
- C++17 标准支持

## 编译步骤

1. 使用 Qt Creator 打开 `final.pro`
2. 配置构建套件（Kit）
3. 点击"构建"按钮编译项目

## 使用说明

1. 在输入框中输入待翻译的单词或句子
2. 选择翻译方向（英→中 或 中→英）
3. 点击"翻译"按钮或按回车键
4. 查看翻译结果和例句
5. 在右侧历史记录区域可以查看、搜索、删除历史记录
